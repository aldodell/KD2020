broadcastLocalMessage()

Broadcast a local message for all or a particular application. Do not use this method to send messages for other networks nodes. Each message has a detinationIdentifier property. If this property is set to "*" all applications will receive the message. In other case only the application with proper identifier will receive it. Each application must override processMessage method in order to handle the message
